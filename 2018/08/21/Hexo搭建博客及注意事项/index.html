Hexo搭建博客及注意事项

发表于 2018-08-21 | 分类于 Hexo
  1. 所有的配置文件的编辑都在sublime中完成，切勿在terminal下使用vim进行修改编辑，否则会出现各种问题。
  2. 切换主题后，使用hexo clean清空，在同一主题下的各种修改都不要使用hexo clean，否则可能出现未知的错误。
    3.hexo g无法生成静态文件，可查看npm缺失的插件npm ls --depth 0,然后根据缺失的插件提示，使用npm install xxx -g即可。
